Architecture Pain Points

  • Ab Initio license: ~$500K-$1M/year for Co>Operating System
  • Teradata appliance: Additional $500K+/year
  • Proprietary GDE — cannot use standard dev tools
  • No Git version control for graph definitions
  • Specialized Ab Initio developers are scarce and expensive
  • 2-4 week change cycle (develop in GDE → test → promote via EME)
  • Data trapped in Teradata — no open format portability
  • Single point of failure — Ab Initio server outage halts ETL

  • Full Git version control for all pipeline code
  • Apache Iceberg — open table format, no vendor lock-in
  • Schema evolution — add/rename columns without data rewrite
  • Time travel — query data as of any historical snapshot
